Before Reporting an Issue

To help us resolve your issue faster, please include:

  • Your operating system and version (e.g., macOS 14, Windows 11, Ubuntu 22.04)
  • XtoMarkdown version (found in the About dialog)
  • The file format you were trying to convert
  • Steps to reproduce the issue
  • Any error messages you received

Frequently Asked Questions

What file formats are supported?

XtoMarkdown supports DOCX, PDF, PPTX, XLSX, HTML, RTF, ODT, EPUB, CSV, JSON, XML, and more. The exact formats depend on which conversion engine you use (Pandoc or MarkItDown).

Why isn't my PDF converting correctly?

PDF conversion quality depends on how the PDF was created. PDFs that were created from text documents generally convert better than scanned documents or image-based PDFs.

Can I convert multiple files at once?

Yes! You can drag and drop multiple files into XtoMarkdown and convert them all at once.

Is XtoMarkdown free?

Yes, XtoMarkdown is free and open-source software licensed under GPL-3.0. You can download it from GitHub Releases or your platform's app store.
